# BİL101Project

Basit bir "Stranger Things" fan sayfası içeren HTML projeleri.

## İçerik
- `HomePage.html` ve çeşitli karakter sayfaları (e.g., `Will.html`, `Eleven.html`, ...)
- `resim/` klasöründe kullanılan görseller

## Nasıl çalıştırılır
1. Bu klasörü bilgisayarınıza klonlayın veya indirin.
2. `HomePage.html` dosyasını tarayıcınızda açın.
